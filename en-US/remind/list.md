# Reminder list

> 如果你想查询抽奖使用的提醒，请参阅：[查询启用的提醒](../roll/remind.md)

```
/remind list
```

> You can only query the alarm you created yourself, see：[提醒的可见范围](overview.md#ti-xing-de-ke-jian-fan-wei)

## Localization & Alias

This command can also be triggered by

```
/ard list
```
