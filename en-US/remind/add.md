# Create Alarm

```
/remind ad
```

## Create Item

### Reminder Type

#### Timed reminders

Reminder at specified time

#### Alert before end

Notify at the specified time before the end of the drawing

#### Reminder interval

Notify after each time interval

## Localization & Alias

This command can also be triggered by

```
/ard add
```
