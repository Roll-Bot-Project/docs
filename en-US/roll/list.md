description: Queries about selected awards within a specified channel
---

# Query Pickup List

```
/roll list [频道id]
```

Channel ID is not required. Leave empty to return the current channel draw

## Localization & Alias

This command can also be triggered by

```
/r list [频道id]
/r girls [频道id]

/ Lump list [频道id]
/ Pick [频道id]
```
