# Exit Award

```
/roll quit <Pick-up number>
```

## Permissions

To quit a draw you must specify the [roll-member.md](../permission/roll-member.md "mention")

## Localization & Alias

This command can also be triggered by

```
/r quit <Pick-up #>
/r q <Pick-up #>

/ Exit Pick-up #>
```
