# 编辑抽奖

```
/roll edit <抽奖编号>
```

> 你可以在任何地方编辑你的抽奖，如果你希望跨平台使用，你应该先通过 `/bind` 指令绑定你的账户。

## 权限

要编辑一个抽奖，你必须为此抽奖的 [抽奖创建人](../permission/roll-creator.md)

## 本地化 & 别称

该指令也可使用如下方式触发

```
/r edit <抽奖编号>

/编辑抽奖 <抽奖编号>
```
