# 为抽奖禁用提醒器

```
/remind disable <抽奖编号> <提醒器编号1> [提醒器编号2] ...
```

你可以一次禁用多个提醒器，提醒器编号之间用空格隔开。

> 抽奖开奖后，将自动禁用所有提醒。

## 权限

要为抽奖禁用提醒器，你必须为 [roll-creator.md](../permission/roll-creator.md "mention") 或 [bot-admin.md](../permission/bot-admin.md "mention")

## 本地化 & 别称

该指令也可使用如下方式触发

```
/rd disable <抽奖编号> <提醒器编号1> [提醒器编号2] ...
```
