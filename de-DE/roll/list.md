---

# Verlosungsliste abrufen

```
/roll list [Kanal-ID]
```

Die Kanal-ID ist optional. Wenn sie leer gelassen wird, wird die Verlosung im aktuellen Kanal zurückgegeben

## Lokalisierung & Alias

Dieser Befehl kann auch wie folgt ausgelöst werden

```
/r list [频道id]
/r ls [频道id]

/抽奖列表 [频道id]
/在抽啥 [频道id]
```
