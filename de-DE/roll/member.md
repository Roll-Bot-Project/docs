# Abfragen der Mitgliederliste

```
/roll Mitglied <Verlosungsnummer>
```

## Lokalisierung & Alias

Dieser Befehl kann auch wie folgt ausgelöst werden

```
/r Mitglied <Verlosungsnummer>
/r mem <Verlosungsnummer>

/Verlosungsmitglied <Verlosungsnummer>
```
