---
description: Interaktive Bearbeitung einer bereits erstellten Verlosung durchführen
---

# Gewinnspiel bearbeiten

```
/roll edit <Verlosungsnummer> [Parameter]
[-o --offener-Bereich ändern] Ändern Sie den offenen Bereich
[-r --Erinnerungsbereich ändern] Ändern Sie den Erinnerungsbereich
[-t --Titel] Ändern Sie den Titel
[-d --Beschreibung] Ändern Sie die ausführliche Beschreibung
[-p --Preis] Ändern Sie den Preis
[-a --automatisches-Ende ändern] Ändern Sie die Option für automatische Verlosung
[-j --Beitritts-Schlüssel ändern]
```
