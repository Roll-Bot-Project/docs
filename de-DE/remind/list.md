# 查询 Erinnerung器列表

> 如果你想查询抽奖使用的提醒，请参阅：[查询启用的提醒](../roll/remind.md)

```
/remind list
```

> 你只能查询你自己创建的 Erinnerung器，详阅：[ Erinnerung的可见范围](overview.md#ti-xing-de-ke-jian-fan-wei)

## Lokalisierung & Alias

Dieser Befehl kann auch wie folgt ausgelöst werden

```
/rd list
```
